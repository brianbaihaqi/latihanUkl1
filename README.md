Program ini dibuat untuk menghitung biaya pengiriman paket. Cara kerjanya adalah dengan meminta pengguna memasukkan berat paket, jarak pengiriman, dan volume barang.
Jika jarak pengiriman kurang dari atau sama dengan 10 kilometer, maka biaya per kilogram adalah Rp 4.250. Jika jaraknya lebih dari 10 kilometer, maka biaya per kilogram adalah Rp 6.000.
Setelah itu, program akan menghitung total biaya berdasarkan berat dan tarif tersebut.
